## Discovery Plan: 하루 1만원 살기 플래너

**날짜**: 2026-03-24
**제품 단계**: 신규 제품
**디스커버리 질문**: "하루 1만원 살기"에 특화된 심플 미니멀 플래너 앱에 대한 제품-시장 적합성이 존재하는가?

**컨셉**: 심플 미니멀 UI, 숫자가 감정을 표현하는 앱

### 실험 상세

#### 실험 1: 랜딩페이지 프리토타입 (V2 검증)

- **가설**: "하루 1만원 살기 플래너" 컨셉을 본 10~20대 중 10% 이상이 사전 등록한다
- **설계**:
  - 미니멀 디자인의 1페이지 랜딩페이지 제작
  - 핵심 기능 3개 소개 (카운트다운, 이월, 원탭 기록)
  - "출시 알림 받기" 버튼 (이메일/카카오톡 수집)
  - 핵심 메시지: "숫자로 느끼는 하루 만원 챌린지"
- **측정**: 방문자 수, 사전등록 수, 전환율
- **판단 기준**:
  - 전환율 ≥ 10% → MVP 개발 진행
  - 전환율 5~10% → 메시지/컨셉 수정 후 재테스트
  - 전환율 < 5% → 컨셉 재검토 또는 피벗

#### 실험 2: 커뮤니티 반응 + 검색 수요 (B1 검증)

- **가설**: 10~20대 커뮤니티에서 긍정 반응 70% 이상 + 관련 검색량 월 1,000건 이상
- **설계**:
  - 에브리타임, 트위터(X), 인스타그램, 에펨코리아 등에 컨셉 이미지 + "만원 살기 도전기" 게시
  - 네이버 키워드 도구/구글 트렌드로 "만원 살기", "짠테크", "하루 절약" 검색량 분석
- **측정**: 좋아요/댓글 긍정 비율, 검색량 데이터
- **판단 기준**:
  - 두 조건 충족 → 충분한 시장 존재, 진행
  - 검색량만 부족 → 시장 교육 필요, GTM 전략 강화
  - 커뮤니티 반응 부족 → 컨셉/타겟 재검토

#### 실험 3: 미니멀 통계 공유 카드 시드 테스트 (G1 검증)

- **가설**: 미니멀 통계 공유 카드의 공유율 5% 이상, 저장율 8% 이상
- **설계**:
  - 미니멀 숫자 중심의 통계 공유 카드 목업 3~5종 제작
  - 예시: "오늘 2,300원 남겼다", "7일 연속 만원 살기 성공"
  - 인스타 스토리/릴스, 트위터에 게시
  - "이런 앱 있으면 쓸 것 같아?" 투표 포함
- **측정**: 공유수, 저장수, 투표 결과
- **판단 기준**:
  - 성공 → 바이럴 루프 기반 성장 전략 채택
  - 실패 → 유료 마케팅 or 커뮤니티 마케팅으로 전환

#### 실험 4: 금액 적절성 설문 (V1 검증)

- **가설**: 10~20대의 60% 이상이 "하루 1만원"을 도전적이면서 현실적인 금액으로 인식
- **설계**:
  - 구글폼/타입폼 설문 (4~5문항)
  - ① 하루 평균 지출 금액
  - ② "1만원으로 하루 살기"가 도전해볼 만한지 (5점 척도)
  - ③ 본인에게 맞는 금액은? (5천/1만/1.5만/2만/직접입력)
  - ④ 이런 앱이 있다면 써볼 의향
- **측정**: 금액별 응답 분포, 사용 의향
- **판단 기준**:
  - "1만원 적절" ≥ 60% → 금액 확정
  - "1만원 적절" < 60% → 사용자 맞춤 금액 설정 기능 추가 검토
