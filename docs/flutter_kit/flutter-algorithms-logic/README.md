# flutter-algorithms-logic

이 스킬은 Flutter/Dart에서 비즈니스 로직, 자료구조 선택, 탐색/정렬, 검증 로직을 효율적으로 설계할 때 사용합니다.

## 언제 사용하는가

- 필터링, 검색, 정렬 로직을 구현할 때
- debounce, throttle, memoization 같은 로직 패턴이 필요할 때
- UI 밖으로 순수 비즈니스 로직을 분리하고 싶을 때
- 검증 로직을 여러 화면에서 재사용해야 할 때

## 핵심 요약

- `Map`, `Set`, `List` 중 접근 패턴에 맞는 자료구조를 선택합니다.
- 불필요한 중첩 루프, 반복 정렬, 반복 스캔을 줄입니다.
- 복잡한 로직은 Widget 안에 넣지 않고 순수 함수나 도메인 서비스로 분리합니다.
- 검증 로직은 재사용 가능하게 중앙화합니다.
