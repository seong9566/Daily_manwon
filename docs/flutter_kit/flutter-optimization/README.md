# flutter-optimization

이 스킬은 Flutter/Dart 런타임 성능 문제를 진단하고 병목 지점을 개선할 때 사용하는 성능 전용 스킬입니다.

## 언제 사용하는가

- 화면이 느리거나 프레임 드랍이 발생할 때
- rebuild가 과도하게 발생할 때
- JSON 파싱, 반복문, 데이터 처리 비용이 커 보일 때
- sqlite 조회나 네트워크 응답 처리로 인해 체감 성능이 떨어질 때

## 핵심 요약

- 먼저 측정하고, 병목을 찾은 뒤, 해당 지점만 최적화합니다.
- `Flutter DevTools`로 CPU, 프레임, 메모리 관점에서 확인합니다.
- `build()` 안의 무거운 작업, 과도한 rebuild, 큰 파싱 작업, 중복 요청을 줄입니다.
- 최적화 후에는 반드시 다시 측정해 효과를 검증합니다.
